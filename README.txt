Welcome to Olin-Man, a Franklin W. Olin College of Engineering implemenation of the classic game Pacman.
Included:
	-An installation guide, to tell you what you need to run this game of pacman
	-A user guide, to tell you how to run this game
	-The code, to play pacman
	-In the folder 'Pictures', a set of pictures of the game.
	-In the folder 'data', images of the ghosts and highscore data.
